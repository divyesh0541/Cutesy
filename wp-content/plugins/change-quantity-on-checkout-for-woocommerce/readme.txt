Allow the customers to change the quantity & delete the product on the checkout page for WooCommerce.

== Description ==

This plugin will allow the customers to change the quantity using WooCommerce standard quantity field & delete the product on the WooCommerce checkout page.

### Change Quantity on Checkout Pro for WooCommerce
The [Change Quantity on Checkout Pro for WooCommerce](https://www.navonmeshsolution.com/) allow user to delete the product on checkout without refreshing the page, additionally you can hide the delete button. 

With Pro version you can add the custom quantity buttons for quantity field, and you add the product thumbnail. Now, you can filter the product category from setting and quantity fields will be displayed respectively. Allow your user to empty the cart from checkout page

###1. Delete the product without refreshing the checkout page
###2. Select the product category for which the quantity field will be displayed
###3. Hide product delete button
###4. Add a custom quantity button for a custom theme
###5. Display product thumbnail
###6. Add an Empty cart button on the checkout page

== Installation ==

1. Ensure you have latest version of WooCommerce plugin installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Quantity selector and Product delete icon will appear on the checkout page of your store.
